# Kovil-Management-System-development-
A Kovil Management System developed using php and SQL server as a web app to manage pooja bookings, priest duties, donations, and temple events.

## 📌 Overview

Product Backlogs 1 and 2 focus on developing the core and extended functionalities of the Kovil Management System. PB1 establishes the foundational features, while PB2 enhances the system with administrative controls, additional modules, and improved user experience.

---

## 🎯 Objectives

* Build core system structure using MVC architecture
* Implement user authentication and booking features
* Introduce admin functionalities and advanced modules
* Improve system usability and interface design

---

## Features Implemented

###  User Authentication (PB1)

* User Registration and Login


###  Pooja Schedule (PB1)

* View available pooja schedules

###  Booking System (PB1)

* Store booking data

---

###  Admin Dashboard (PB2)

* Manage system operations
* View and control bookings

###  UI Enhancements (PB2)

* Improved layout and styling
* Better user experience

---

##  Technologies Used

* PHP (MVC Architecture)
* MySQL Database
* HTML/CSS

---


##  Key Files

* Models: `User.php`, `Schedule.php`, `Login.php`, `Register.php`,
* Views: Authentication, Schedule, Booking, Admin, Duty, Payment modules

---

## Conclusion

PB1 and PB2 collectively deliver a functional and structured system, starting from core user features and progressing to advanced administrative capabilities, ensuring a scalable and user-friendly Kovil Management System.
